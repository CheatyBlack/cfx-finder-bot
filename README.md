## <ins>Installation

1. Clone the repository (`git clone https://github.com/alejandromume/cfx-finder-bot` )
2. Install the dependencies (`npm i`)
3. Open the `index.js` and add the bot token
4. Run the bot (`node .` or `node index.js`)
